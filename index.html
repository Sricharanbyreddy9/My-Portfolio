RAGCare - Retrieval-Augmented Healthcare Assistant

RAGCare is an intelligent, document-aware chatbot built using Flask that leverages Retrieval-Augmented Generation (RAG) to provide accurate, context-based medical assistance.

Core Functionality

  • Document Upload & Processing: Users can upload medical documents (PDFs like reports, prescriptions, health articles). These documents are automatically chunked and processed.
  • Vector Storage: Processed text chunks are embedded using SentenceTransformers and stored in a FAISS vector index for efficient similarity search.
  • Intelligent Retrieval: When a user asks a question, RAGCare retrieves the most relevant text chunks from the uploaded or preloaded documents.
  • Contextual Generation: The retrieved context is combined with the user's query and passed to a Large Language Model (LLM) like GPT or LLaMA (via Groq API) to generate a grounded, informative response.

Key Features

  • Support for multiple LLM backends (OpenAI, Groq LLaMA).
  • Chat history and export functionality.
  • PDF preview capability.
  • Modern UI with responsive design and dark mode.
  • Demonstrates the power of GenAI and RAG pipelines in bridging unstructured medical documents with conversational AI.
